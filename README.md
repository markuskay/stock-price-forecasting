# Stock Price Forecasting Using LSTM

A deep learning system for forecasting stock prices using Long 
Short-Term Memory (LSTM) networks.

## Overview
This study develops and evaluates an LSTM-based deep learning model 
for predicting stock market prices by learning temporal patterns and 
dependencies from historical stock data. The model captures long-range 
sequential relationships in financial time series to generate reliable 
short and medium term price forecasts.

## Model Architecture
- Long Short-Term Memory (LSTM) network
- Designed to capture temporal dependencies in sequential financial data
- Trained on historical stock price data

## Evaluation Metrics
- Accuracy, RMSE, MAE, R-squared

## Tech Stack
- Python
- TensorFlow
- Scikit-learn
- Streamlit
- Pandas, NumPy, Matplotlib
